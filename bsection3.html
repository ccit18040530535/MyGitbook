2.3 hexo搭建

(1)安装**hexo：在cmd窗口使用下面的命令

npm install hexo-cli -g

若结果以下结果，则表示安装成功

还可输入以下指令来进行核查，

hexo -v

若结果显示版本号，则表示安装成功

(2) 创建hexo博客目录

md blog

(3) 初始化hexo博客：

依次执行下面两条命令

cd blog
hexo init

若实验结果出现“Start blogging with Hexo!”字符串，则表示初始化成功且可以开始写博客了。
